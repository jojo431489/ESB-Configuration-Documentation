整體概覽

本文件說明 ESB (Enterprise Service Bus) 平台的設定檔結構與內容。ESB 平台用於整合 ERP 系統與 MES 系統之間的資料交換。

📁 檔案清單

  • DataModel.json資料模型
  • ProductService.json服務定義
  • ProductServiceDetail.json完整設定

🏷️ 服務類型分布

  • MES47 個服務
  • WFGP11 個服務
  • YIFEI22 個服務

架構說明

🔄 資料流向

  • 1. ERP 資料擷取SOAP WebService
  • 2. 欄位映射轉換ESB 平台處理
  • 3. 資料推送 MESREST API

📋 支援協議

  • SOAP WebServiceERP 整合
  • REST APIMES 整合
  • HTTP POST通用

檔案結構說明

每個 JSON 檔案都包含以下標準結構：

節點名稱 說明 用途
monJob 監控作業 定義排程監控任務
dataModel 資料模型 定義資料表結構與 SQL 腳本
dataModelField 資料模型欄位 定義每個資料模型的欄位屬性
dataModelRelate 資料模型關聯 定義資料模型之間的 JOIN 關係
productService 產品服務 定義 Web Service 或 API 端點
productServiceDetail 服務欄位詳情 定義服務的輸入/輸出欄位映射
conditionGroup 條件群組 定義資料過濾條件
dbSetting 資料庫設定 定義資料庫連線資訊
